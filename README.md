# ImageDotGene
## 概要
Openframeworksで開発。画像をドット化する
## 使い方
1. 起動時にダイアログが開くので，ドット化したい画像を選択
2. ドット化されたものが画面に出力される
3. キーボード入力で色々パラメータ設定が可能
  - 'X'
    - 横幅のドット数を増やす
  - 'x'
    - 横幅のドット数を減らす
  - 'Y'
    - 縦幅のドット数を増やす
  - 'y'
    - 縦幅のドット数を減らす
  - 'l'
    - ドットのサイズを大きくする
  - 's'
    - ドットのサイズを小さくする
  - 'c'
    - ドットの数・サイズなどをデフォルトに戻す
  - 'o'
    - 画像選択ダイアログが開く
  - 'p'
    - 現在の画像をカレントディレクトリに"saveImage.jpg"として書き出す
